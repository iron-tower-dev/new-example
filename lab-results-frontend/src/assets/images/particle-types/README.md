# Particle Type Images

This directory contains reference images for particle types used in Inspect Filter and Ferrography tests.

## Image Files

The following image files should be placed in this directory:

### Rubbing Wear
- rwp1.jpg, rwp2.jpg (Rubbing Wear platlet)
- rw1.jpg, rw2.jpg (Rubbing Wear)

### Oxides
- bo1.jpg, bo2.jpg (Black Oxide)
- dmo1.jpg, dmo2.jpg (Dark Metallo-Oxide)
- ro1.jpg, ro2.jpg (Red Oxide/Rust)

### Wear Particles
- aw1.jpg, aw2.jpg (Abrasive Wear)
- re1.jpg, re2.jpg (Rework)
- swp1.jpg, swp2.jpg (Severe Wear Particles)
- ch1.jpg, ch2.jpg (Chunks)
- sp1.jpg, sp2.jpg (Spheres)

### Other Materials
- nfm1.jpg, nfm2.jpg (Non Ferrous Metal)
- co1.jpg, co2.jpg (Corrosive)
- nmc1.jpg, nmc2.jpg (Non Metallic Crystalline)
- nma1.jpg, nma2.jpg (Non Metallic Amorphous)
- fp1.jpg, fp2.jpg (Friction Polymer)
- fi1.jpg, fi2.jpg (Fibers)

## Image Requirements

- Format: JPG
- Size: Recommended 200x200 pixels or similar square aspect ratio
- Quality: High enough to show particle characteristics clearly
- Background: Preferably consistent lighting and background

## Fallback

If images are not available, the component will show a placeholder or broken image icon.
